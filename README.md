# 简单会员管理系统

这是一个使用C++开发的简单会员管理系统，具有以下功能：

- 会员注册
- 会员信息查看
- 会员类型管理（Gold/Silver）
- 消费积分计算

## 系统要求

- Visual Studio 2022或更高版本
- EasyX图形库
- MySQL数据库

## 安装步骤

1. 克隆项目到本地
2. 使用Visual Studio打开Project1.sln
3. 配置MySQL数据库连接信息
4. 编译并运行项目

## 使用说明

1. 启动程序后，会显示主界面
2. 点击"添加会员"按钮可以注册新会员
3. 点击"查看会员"按钮可以查看所有会员信息
4. 点击"退出系统"按钮可以关闭程序

## 注意事项

- 会员ID必须是6位数字
- 会员类型分为Gold和Silver两种
- 消费满400元自动升级为Gold会员
- 每消费1元可获得10积分
